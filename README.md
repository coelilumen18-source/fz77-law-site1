# Свод законов: ФЗ-77

1. Создайте репозиторий на GitHub.
2. Загрузите файлы.
3. В Settings > Pages выберите branch `main` — сайт на https://yourusername.github.io/repo.
4. Замените `your_bot` в HTML/JS на вашего TG-бота с Mini App.
5. Дополните полный текст статей из официальных источников (pravo.gov.ru).

Сайт responsive, с аккордеоном как в Консультанте, градиентами и темами.
